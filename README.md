# Picoシリーズ/MicroPythonによる日本標準電波放送（JJY）サンプル群

日経BPパソコンベストムック 「[ラズパイ工作パーツ大全2026](https://info.nikkeibp.co.jp/media/RAS/atcl/books/031400029/)」に掲載した電波時計と、電波時計の制作に利用した疑似JJY送信機のソースコード一式です。ソースコードや使い方の詳細は、「[ラズパイ工作パーツ大全2026](https://info.nikkeibp.co.jp/media/RAS/atcl/books/031400029/)」を参照してください。

## ディレクトリ構成

* **PseudoJJY/**: 疑似JJY送信機（JJY開発支援ツール）
* **WaveClock/**: 電波時計
